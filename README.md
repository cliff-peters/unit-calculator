# 8-Minute Rule Unit Calculator

A CMS 8-minute rule calculator for Medicare Part B 
therapy billing. Built for outpatient and assisted 
living therapy teams.

## Features
- Multi-intervention calculator with individual time entries
- Total time calculator with quick preset buttons
- Minutes to next billable unit with progress indicator
- Full unit reference table

## Usage
Enter timed intervention minutes to calculate billable 
units and see how close you are to the next threshold.
