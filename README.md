# Unit Calc

8-minute rule unit calculator for therapy billing in assisted living.

Live: [https://cliff-peters.github.io/unit-calculator](https://cliff-peters.github.io/unit-calculator)

## Tabs

**Total Minutes** — Enter a single total minute value for a quick unit lookup. Includes preset buttons for common treatment times.

**Multi-Intervention** — Enter each timed intervention with its name and minutes. Totals your minutes, calculates billable units, and shows a full breakdown.

## Features

- Billable unit calculation with color-coded results
- Minutes to next unit with progress bar and proximity alerts
- Full 8-minute rule reference table with current unit highlighted
- Intervention breakdown for multi-intervention sessions
- Single HTML file, no dependencies
